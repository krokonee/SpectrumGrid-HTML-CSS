# SpectrumDrid with HTML/CSS

A web project with a responsive mosaic block layout styled using HTML and CSS Flexbox.

# Installation / How to Run

1. Download or unzipp the project folder.
2. Open 'fleex.html' in your browser.
   
# Usage

* Open the page to view the interactive mosaic grid.
* Hover over any colored tile to see smooth micro-interactions and depth animations.
  
# Features

* 🧩 **Bento Grid Layout:** A modern, visual grid structure inspired by the trending Bento UI style.
* 🎨 **Vibrant Gradients:** Beautifully curated color palette with custom linear gradients for each block.
* ✨ **Smooth Animations:** Interactive hover effects featuring subtle scaling, lift, and shadow transitions.
* 📱 **Responsive Design:** Adaptive Flexbox configuration that wraps and adjusts cleanly across various screen dimensions.


# Technologies

* HTML
* CSS (Flexbox, CSS Gradients, Transitions)
